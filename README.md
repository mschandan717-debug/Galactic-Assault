# 🚀 Galactic Assault

Galactic Assault is a fast-paced 2D arcade-style space shooter built using **HTML, CSS, and JavaScript**. Pilot a futuristic spaceship, battle waves of enemy alien ships, dodge incoming attacks, collect power-ups, and survive increasingly difficult levels.

---

## 🎮 Features

- 🚀 Smooth spaceship movement
- 🔫 Laser shooting mechanics
- 👾 Multiple enemy types with unique behaviors
- 💥 Explosion particle effects
- ✨ Glowing laser animations
- ⭐ Animated scrolling starfield background
- ❤️ Health system
- 🏆 Real-time score tracking
- ⚡ Power-up collection
- 🎯 Mouse and keyboard controls
- 🔄 Game Over and Restart functionality
- 📱 Responsive browser gameplay

---

## 🕹️ Controls

| Action | Control |
|--------|----------|
| Move Spaceship | Arrow Keys / WASD / Mouse |
| Fire Lasers | Spacebar / Left Mouse Button |
